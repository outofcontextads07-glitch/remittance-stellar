# remittance-stellar
Remittance Smart Contract (Stellar Testnet)
This repository contains a remittance smart contract deployed on the Stellar testnet. It is designed to experiment with sending funds across borders using programmable logic on Stellar’s Soroban smart contracts.

Overview
The contract is deployed as a WASM smart contract on the Stellar testnet and can be interacted with via standard Soroban RPC and SDKs.

Typical remittance flows you can implement or extend with this contract:

Sender locks or transfers funds under specific rules (beneficiary, amount, optional metadata).

Receiver (beneficiary) claims funds on testnet according to those rules.

Off-chain components (agents, frontends, or oracles) map real-world remittance use cases (e.g., INR payout, KYC checks) to on-chain calls.
